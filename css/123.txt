<!DOCTYPE html> = 항상 쓰는 코드
<html> = 항상 쓰는 코드
<head> = 콘텐츠를 보조하는 내용
<meta charset = "UTF-8"> = 콘텐츠에 한글이 있다는 뜻, 한글쓰면 써줘야 함
<title>김멋사의 이력서</title> = 문서의 제목
<link rel="stylesheet" href="codelion.css"> = 이 문서를 꾸미기 위해서 codelion.css를 참고하겠다 
<body> = 콘텐츠 내용
<h1> = 가장 크게 텍스트 표현, 숫자 커질수록 크기 작아짐
<p> = 모든 텍스트
<div> = 씌우기. p로 모두 표현하기 어려움
<section>, <div> 똑같음. html 요소를 꾸미기 위해 묶는 태그

 <p class="name-text">HTML/CSS 개발자</p>
꾸며주기 위해서 class로 이름 지정함
.name-text {
    font-size:16px;
    color:#7c7c7c;
    font-weight: bold; }

line-height = 줄 간 간격

margin-right : auto; 
margin-left : auto;
박스가 가운데 정렬이 됌

box-shadow : 그림자 
box-shadow: 0 1px 20px 0 rgba(0, 0, 0, 0.1);
r= red g = green b = blue a = 투명도

padding = 박스 안
margin = 박스 밖에 옷입는다고 생각하면 쉬움
padding left = 들여쓰기 효과

float로 한줄 좌우 정렬 가능
overflow : hidden 플로트를 묶어주면 다른 요소랑 겹치지 않음


폰트 가져오는 방법
@import url('');

* { font-family: ''; }
* : 전체 적용

@import url('https://fonts.googleapis.com/css?family=Montserrat:100,200,300,400,500,600,700,800&display=swap');

* {
    font-family: 'Montserrat';
}
